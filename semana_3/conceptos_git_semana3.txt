
Repositorio
Es el espacio virtual o carpeta donde Git almacena y organiza todos los archivos del proyecto.
Guarda el historial completo de los cambios realizados, permitiéndo regresar a versiones anteriores en cualquier momento.
Puede estar guardado de forma local en tu computadora o de forma remota en servidores de internet.


Commit
Es como tipo de fotografía o captura del estado actual de mis archivos en un momento específico.
 Cada vez que se realiza este paso,se  guarda de forma permanente los cambios que has hecho en el historial del proyecto. 
 Cada uno incluye un mensaje descriptivo que explica qué se modificó y un identificador único.


Push

Es la acción de subir y enviar todos los cambios locales que se guarda mediante los commits hacia un servidor remoto.
 Permite actualizar el repositorio compartido en plataformas como GitHub, GitLab o Bitbucket para que otros puedan ver tu trabajo.
 Es la herramienta fundamental para mantener sincronizado tu avance con el resto de tu equipo
